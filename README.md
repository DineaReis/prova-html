 Calendário HTML

Este projeto foi desenvolvido como atividade prática de HTML.

Descrição
O projeto consiste na criação de um calendário com três meses utilizando tabelas em HTML.

Tecnologias utilizadas
- HTML
- CSS

Estrutura do projeto
- index.html → página principal do calendário
- style.css → arquivo de estilos


Funcionalidades
- Calendário de três meses
- Destaque de feriados
- Inclusão de atividades e agendamentos
